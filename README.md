# Age Calculator App

A simple and responsive age calculator built with HTML, CSS, and JavaScript. Users can enter their birth date to calculate their exact age in years, months, and days.

## Features

- Calculate age in years, months, and days
- Validate empty fields
- Validate invalid dates
- Prevent future dates
- Responsive design for desktop and mobile devices
- User-friendly error messages

## Built With

- HTML5
- CSS3
- JavaScript (ES6)

## Getting Started

1. Clone the repository

```bash
git clone <repository-url>
```

2. Navigate to the project directory

```bash
cd age-calculator-app
```

3. Open `index.html` in your browser.

Alternatively, if you use VS Code, install the **Live Server** extension and click **Go Live**.

## Validation Rules

The application checks that:

- All fields are filled in
- The month is between 1 and 12
- The day is valid for the selected month and year
- The selected date is not in the future
